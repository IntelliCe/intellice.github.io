MusicPlaybackQueueManager

class MusicPlaybackQueueManager @PackagePrivate constructor

音乐播放队列管理器，该类只会在 MusicPlaybackService 中实例化并管理音乐播放队列。 该类包含以下可供外部调用的操作：

  • switchPlaybackMode — 切换播放模式

  • setPlaybackQueue — 设置播放列表

  • append — 将音乐添加到播放列表

  • remove — 将音乐移除播放列表

  • clearPlaybackQueue — 清空播放列表

  • addListener/removeListener — 添加或移除监听器
